## âš ï¸ Troubleshooting

### Music Doesn't Play?
- Make sure the file is named exactly `music.mp3`
- Try clicking on the page (some browsers block autoplay)
- Check the file size (keep under 5MB)

### Photos Not Showing?
- Check file names match exactly (case-sensitive!)
- Make sure photos are in the same folder as index.html
- Try .jpg format (not .jpeg or .png)

### Website Not Loading?
- Wait 2-3 minutes after enabling GitHub Pages
- Clear your browser cache (Ctrl+F5)
- Check if repository is Public (not Private)

### Eyes Not Tracking?
- Only works on desktop/laptop with mouse
- Make sure you're on the landing page
- Try a different browser (Chrome/Firefox)
